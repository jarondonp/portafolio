JRYOLO - Aplicación de Visión Artificial con YOLO

Resumen del Proyecto

JRYOLO es una aplicación web completa para el procesamiento de imágenes y videos mediante modelos YOLO (You Only Look Once), que proporciona una interfaz amigable para realizar diversas tareas de visión artificial.

Visión Artificial Python/Flask YOLO Web App OpenCV
Ver en GitHub

Estructura Principal

La aplicación está organizada como un servicio web basado en Flask con una estructura modular:

  • Backend (Python/Flask): Maneja la lógica de procesamiento de imágenes/videos y el entrenamiento de modelos
  • Frontend (HTML/CSS/JavaScript): Ofrece interfaces intuitivas para diferentes funcionalidades
  • Módulos de Utilidades: Componentes especializados que gestionan aspectos específicos
    • yaml_utils.py: Manejo robusto de archivos YAML (corrige errores comunes)
    • model_utils.py: Extracción de información de modelos YOLO desde múltiples fuentes
    • metrics_utils.py: Gestión de métricas de modelos con capacidad para buscar en múltiples ubicaciones

Funcionalidades Principales

1. Detección de Objetos

  • Permite cargar imágenes para identificar y localizar objetos
  • Proporciona ajuste de umbral de confianza
  • Visualiza los resultados con cuadros delimitadores
  • Soporta procesamiento de imágenes desde URLs

2. Segmentación de Imágenes

  • Segmentación semántica a nivel de píxel de objetos en imágenes
  • Visualización con máscaras transparentes coloreadas
  • Interface especializada para este tipo de procesamiento

3. Estimación de Poses

  • Detecta puntos clave del cuerpo humano en imágenes
  • Visualiza esqueletos y articulaciones sobre las personas detectadas

4. Clasificación de Imágenes

  • Identifica la categoría principal de una imagen
  • Muestra probabilidades para diferentes clases posibles

5. Análisis de Video

  • Procesamiento de archivos de video subidos por el usuario
  • Streaming en tiempo real desde webcam
  • Visualización de resultados de detección/segmentación/pose en video
  • Captura de fotogramas individuales para análisis

6. Entrenamiento de Modelos

  • Interface completa para entrenar modelos YOLO personalizados
  • Soporte para cargar datasets en formato ZIP
  • Verificación automática de la estructura del dataset
  • Configuración de parámetros de entrenamiento (épocas, batch size, etc.)
  • Seguimiento del progreso de entrenamiento en tiempo real
  • Visualización de métricas de rendimiento

7. Gestión de Modelos

  • Visualización detallada de las clases que puede detectar el modelo
  • Exploración de métricas de rendimiento
  • Acceso a artefactos de entrenamiento y visualizaciones
  • Documentación automática del modelo

Características Técnicas

  • Compatibilidad con múltiples formatos:
    • Imágenes: PNG, JPG, JPEG, GIF
    • Videos: MP4, AVI
    • Datasets: ZIP
  • API RESTful:
    • Endpoints para todas las funcionalidades principales
    • Respuestas en formato JSON para integración con otras aplicaciones
  • Procesamiento asíncrono:
    • Entrenamiento en hilos separados para no bloquear la interfaz
    • Monitoreo en tiempo real del progreso de operaciones largas
  • Almacenamiento organizado:
    • Estructura de carpetas para uploads, resultados, modelos y datasets
    • Generación de nombres únicos para archivos
  • Compatibilidad con Hugging Face Spaces:
    • Adaptaciones para ejecutarse en entornos cloud
    • Puerto configurado para 7860
  • Manejo Robusto de Errores:
    • Verificación exhaustiva de datos de entrada
    • Recuperación de errores comunes en archivos de configuración
    • Logging detallado para diagnóstico

La aplicación ha sido diseñada para ser accesible tanto para usuarios con conocimientos técnicos limitados como para profesionales de visión artificial, ofreciendo una interfaz intuitiva pero potente para trabajar con modelos YOLO.
