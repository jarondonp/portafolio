AI Executive Agents Ecosystem in n8n

AI Agents Automation n8n LangChain API Integration

A comprehensive ecosystem of intelligent agents working together to provide a complete automation solution for managing communications, scheduling, and information retrieval through natural language interactions.

Executive Agent JR

Main user interaction agent through Telegram that coordinates multiple productivity tools using AI, processes both text and voice inputs, and maintains conversation memory.

Email Agent

Automates email management through Gmail API with capabilities for sending, replying, labeling, and organizing messages with a natural human-like tone.

Calendar Agent

Manages Google Calendar events with features for creating, editing, and deleting appointments, including automatic Google Meet link generation and attendee management.

Technologies

  • n8n for workflow automation and orchestration
  • LangChain Tools Agent framework
  • OpenAI Chat Models for natural language understanding
  • Anthropic Claude for human-like email composition
  • Google APIs (Gmail, Calendar, Sheets)
  • SerpAPI for real-time information retrieval
  • Telegram Bot API
  • Speech-to-Text for voice note processing

Implementation

Key technical aspects of this project:

  • Agent orchestration through n8n workflows
  • LangChain Tools for decision making and task routing
  • API integrations with Google services
  • Simple Memory system for session persistence
  • JSON parsing for structured data exchange
  • Conditional logic for adaptive response handling

Agent Capabilities

Executive Agent JR

  • Tools:
    • contact_tool: Query contacts in Google Sheets
    • contact_tool_update: Create/update contacts
    • calendarTool: Connect to Calendar Agent
    • emailTool: Connect to Email Agent
    • serpapi: Search for news and updated data
    • calculator: Solve formulas
  • Input: Text and voice through Telegram
  • Output: Formatted response + tool execution in Telegram
  • Memory: Per session via Simple Memory

Email Agent

  • Tools:
    • Send Email: Send new emails
    • Email Reply: Reply to emails
    • Label Emails: Tag messages
    • Delete Message: Remove emails
    • Create Draft: Save drafts
    • Get Emails: Retrieve messages
    • Get Labels: List tags
    • Mark Unread: Mark as unread
    • contact_tool: Check contacts before sending
  • Entry point: Called by the Executive Agent
  • Output: Success or retry notification

Calendar Agent

  • Tools:
    • Create Event: Schedule new events
    • Create Event with Attendee: Include participants
    • Create Meet: Generate Google Meet link
    • Update Event: Modify existing events
    • Delete Event: Remove events
    • Get Events: List upcoming events
    • contact_tool: Verify participant information
  • Entry point: Called by the Executive Agent
  • Output: Success or error status
